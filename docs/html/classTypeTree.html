Grows the typetree in a deepth-first manner. The initial call should be called with the root node, an all-empty status array, the middle index of the status array, move=-1, connections set to 0 and blocks set to false.

Parameters
nodeThe current node.
statusThe stone status array
locationThe current position of the status array being branched
moveThe direction of the next move. -1 or 1
blackConnectThe number of connected black stones on the side currently being analyzed. It is used to prevent generating a 5, for example : OOOOO*OOX– ; –X *OOOOO
whiteConnectSame as blackConnect
blackBlockTrue if the side currently being analyzed is already blocked by black stone.
whiteBlockSame as blackBlock
